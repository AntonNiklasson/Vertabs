Welcome to Vertabs!

This is a Chrome Extension for tab junkies. Vertical tabs. Simple as that.

  1. Activate Vertabs (in Chromes top right corner).
  2. Alter the options.
  3. Previously open tabs need to be refreshed.
  4. Close this page.

Chrome extensions are not allowed in pages like Chrome bookmarks, the extensions settings page etc. This page included. It will work in all other tabs.
