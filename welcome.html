Welcome to Vertabs!

This is a Chrome Extension for tab junkies. Vertical tabs. Simple as that.

Chrome extensions are not allowed in pages like Chrome bookmarks, the extensions settings page etc. This page included. It will work in all other tabs.

How to get started:

  1. To activate Chrome Extensions in previously open tabs they need to be refreshed.
  2. Activate Vertabs in Chromes top right corner.
  3. Alter the options if you want to.
  4. Vertabs appears on your left hand side. Hover it to reveal the tab list.
  5. Close this page and try to 200 tabs.
