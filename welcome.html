Welcome to Vertabs!

Vertical tabs. Simple as that.

  1. Activate Vertabs (in Chromes top right corner).
  2. Make it yours.
  3. Refresh previously open tabs.
  4. Enjoy vertical tabs.

Chrome extensions are not allowed in pages like Chrome bookmarks, the extensions settings page and so on. This page included.
